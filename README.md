# Restaurant Management System

A web-based food ordering system built with PHP and MySQL. Customers can browse the menu, add items to cart, and place orders online.

## Features
- **Menu Display**: View food items with images and prices
- **Shopping Cart**: Add/remove items and update quantities
- **Checkout System**: Place orders with customer details
- **Admin Panel**: Manage orders and menu items
- **Database Integration**: All data stored in MySQL

## Tech Stack
- **Backend**: PHP
- **Frontend**: HTML, CSS
- **Database**: MySQL
- **Server**: XAMPP/Laragon
